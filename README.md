# 실시간 STT 자막 웹앱

이 폴더는 노트북, 태블릿, 휴대폰 브라우저에서 실행되는 실시간 STT 자막 웹앱을
연습용으로 만들어 보는 공간입니다.

## 목표

- 마이크 음성을 실시간으로 텍스트로 변환하기
- TV에서도 잘 보이도록 큰 자막 화면 만들기
- HTML, CSS, JavaScript만 사용해서 단순하게 시작하기
- 로컬 서버에서 실행하고 브라우저로 확인하기

## 현재 기능

- 시작 버튼
- 중지 버튼
- 현재 인식 중인 문장을 크게 보여주는 영역
- 최근에 인식된 문장 목록
- 예기치 않은 음성 인식 종료 시 자동 재연결
- 네트워크, 권한, 마이크 등 마지막 오류 표시
- 화면을 다시 열었을 때 음성 인식 자동 복구
- 시스템 기본 마이크 이름과 외부 마이크 후보 표시
- 마이크 입력 레벨 표시
- 자동 재연결 횟수 표시

## 실행 방식

이 폴더에서 로컬 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서는 아래 주소로 접속합니다.

```text
http://localhost:8000
```

마이크 목록과 입력 레벨 확인 기능은 브라우저 보안 정책상 HTTPS 또는
`localhost`에서만 동작할 수 있습니다. 다른 기기에서 네트워크 주소로 접속할
때는 HTTPS를 사용하는 것이 좋습니다.

## 마이크 표시의 의미

- `기본 입력 추정`은 별도 마이크 진단 스트림에서 확인한 시스템 기본 입력입니다.
- Web Speech API가 실제로 사용하는 마이크 이름은 브라우저가 직접 제공하지 않습니다.
- 외부 마이크를 연결한 뒤 앱을 새로고침하고, 기기의 시스템 기본 입력도 확인하는 것이
  가장 확실합니다.
