# 실시간 STT 자막 웹앱

이 폴더는 맥북 로컬에서 실행되는 실시간 STT 자막 웹앱을 연습용으로 만들어 보는 공간입니다.

## 목표

- 마이크 음성을 실시간으로 텍스트로 변환하기
- TV에서도 잘 보이도록 큰 자막 화면 만들기
- HTML, CSS, JavaScript만 사용해서 단순하게 시작하기
- 로컬 서버에서 실행하고 브라우저로 확인하기

## 첫 번째 버전에서 만들 기능

- 시작 버튼
- 중지 버튼
- 현재 인식 중인 문장을 크게 보여주는 영역
- 최근에 인식된 문장 목록

## 다음에 Codex에게 요청할 일

```text
이 폴더에 HTML, CSS, JavaScript만 사용해서 실시간 STT 자막 웹앱 첫 버전을 만들어줘.
index.html, style.css, script.js 파일로 구성하고,
시작/중지 버튼, 큰 자막 표시 영역, 최근 문장 목록을 포함해줘.
```

## 로컬 실행 예정 방식

앱 파일이 만들어진 뒤에는 이 폴더에서 로컬 서버를 실행할 예정입니다.

```bash
python3 -m http.server 8000
```

브라우저에서는 아래 주소로 접속합니다.

```text
http://localhost:8000
```
